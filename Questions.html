Q1. Create an arrow function called square that takes a number as an argument and returns its square. Use the arrow function to calculate the square of a given number and display the result.

code:-
function square(num) {
const square = (num) => num * num;
return num*num; }
const number = 5;
const result = square(number);
console.log("The square of", number, "is", result);

Explanation:-
  • (num) is the parameter.
  • => indicates that this is an arrow function.
  • num * num is the expression that calculates the square of the input num.

    • Q3. Create an IIFE (Immediately Invoked Function Expression) that calculates the square of a number and immediately displays the result.

    code:-
    (function(num) {
    const result = num * num;
    console.log(`The square of ${num} is ${result}`);
    })(10);

    Explanation:-
    IIFE Syntax:
  • The function is defined and immediately invoked by wrapping it in parentheses (function(){...}) and following it with another set of parentheses () containing the argument.
  • num Parameter:- The IIFE takes a number num as an argument.
  • Calculation:- It calculates the square of num using num * num.
  • Immediate Execution:- The function is immediately executed with the argument 10 passed in.

    • Q5. Write a JavaScript function called factorial that calculates the factorial of a non-negative integer using recursion. Test the function with different inputs.

    code:-
    function factorial(n) {
    if (n === 0 || n === 1) {
    return 1; }
    return n * factorial(n - 1); }
    console.log(`Factorial of 0 is ${factorial(0)}`);
    console.log(`Factorial of 1 is ${factorial(1)}`);
    console.log(`Factorial of 5 is ${factorial(5)}`);
    console.log(`Factorial of 7 is ${factorial(7)}`);
    console.log(`Factorial of 10 is ${factorial(10)}`);

    Explanation:-
    1. Base Case:-
  • The base case checks if n is 0 or 1. If it is, the function returns 1 because the factorial of 0 and 1 is 1.
    • 2.Recursive Case:-
  • If n is greater than 1, the function returns n * factorial(n - 1). This means the function calls itself with a decremented value of n, gradually reducing it until it reaches the base case.

    • Q6. Write a JavaScript function called curry that takes a function as an argument and returns a curried version of that function. The curried function should accept arguments one at a time and return a new function until all arguments are provided. Then, it should execute the original function with all arguments. Test the curry function with a function that adds two numbers.

    code:-
    function curry(func) {
    return function curried(...args) {
    if (args.length >= func.length) {
    return func.apply(this, args); }
    else { return function(...nextArgs) {
    return curried.apply(this, args.concat(nextArgs)); };
    } };
    }
    Test:-
    function add(a, b) {
    return a + b; }
    const curriedAdd = curry(add);
    console.log(curriedAdd(5)(3));
    console.log(curriedAdd(10)(20));

    Explanation:-
    1. curry Function:-
  • Input: Takes a function func as an argument.
  • curried Function:-
  • If the number of arguments provided (args.length) is equal to or greater than the number of parameters expected by the original function (func.length), it immediately calls the original function with all the arguments.
  • If not all arguments are provided, it returns a new function that accepts the remaining arguments. This process continues until all required arguments are collected.
    • 2. add Function:
  • A simple function that takes two arguments, a and b, and returns their sum.
    • 3. curriedAdd:
  • The curried version of the add function. It can be called with arguments one at a time.
